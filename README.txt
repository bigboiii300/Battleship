Обозначения:
0-.. - разметка по краям поля (начинаются с 0, следовательно координаты тоже могут быть равны 0)
. - клетка поля
* - был произведен выстрел, но игрок попал мимо
# - был произведен выстрел и игрок попал в корабль

Ввод игрока:
1. Вввод имени: Вводится имя игрока.
2. Ввод размеров поля боя: Ввод двух чисел через пробел. В случае некоректного ввода - повтор.
3. Ввод количества кораблей разного типа: Построчный ввод количества кораблей. В случае некорректного ввода - возвращаемся к п.2
4. Ввод координат (x and y): Ввод двух чисел через пробел. В случае некорректного ввода выстрел не производится и игрок повторяет ввод.
5. Окончание игры и возможность рестарта или выхода.

Команды после завершения игры:
1. Ввод команды restart - игра начинается заново.
2. Ввод команды exit - происходит выход из программы
